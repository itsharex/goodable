## 🚀 快速开始

### 前提条件
- ✅ Windows系统（Python 3.11+）
- ✅ 已安装并登录微信PC客户端

### 三步启动

**第1步：安装依赖**
```bash
cd project
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt
```

**第2步：配置环境变量（可选）**
```bash
# 如需使用AI总结功能，在 project/.env 中配置
OPENAI_API_KEY=your_api_key_here
```

**第3步：启动应用**
```bash
# 在 project 目录下执行
.venv\Scripts\python -m uvicorn app.main:app --host 127.0.0.1 --port 8000
```

✅ **访问地址：** http://127.0.0.1:8000

> 💡 提示：
> - 启动前确保微信PC客户端已启动并登录
> - wxauto 模块已内置在项目中，无需额外安装
> - 首次启动会自动检测微信窗口并建立连接

## ⚖️ 技术与合规说明

### 技术原理
本工具基于**RPA（机器人流程自动化）**技术，通过模拟用户操作与微信PC客户端交互，实现消息读取和群管理功能。

### 合规承诺
- ✅ **仅读取权限**：只读取本地微信数据，不修改、删除任何内容
- ✅ **本地存储**：所有数据存储在本地SQLite数据库，不上传云端
- ✅ **开源透明**：代码完全开源，欢迎审计
- ✅ **用户授权**：需用户主动启动，不后台运行
- ⚠️ **使用限制**：仅供个人学习研究，不得用于商业用途或违反微信用户协议

### 技术栈
- **后端**：Python 3.11+ / FastAPI / SQLite
- **微信集成**：wxauto 模块（内置，无需额外安装）
- **AI能力**：所有OpenAI 兼容API（可选）
- **前端**：原生HTML/CSS/JavaScript

### 项目结构
```
goodqunbot/
├── wxauto/              # 微信自动化模块（pyc字节码）
├── project/             # 主项目目录
│   ├── app/            # 应用代码
│   │   ├── main.py    # FastAPI 主程序
│   │   ├── wx_manager.py  # 微信管理器
│   │   └── ...
│   ├── .venv/          # 虚拟环境
│   ├── requirements.txt # 项目依赖
│   └── python_dev.db   # SQLite 数据库
└── README.md
```
