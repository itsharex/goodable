# 配置文件说明

## stopwords.txt - 中文停用词表

### 用途
用于词云生成时过滤无意义的高频词，提升词云质量。

### 文件来源
- 基础词表：百度停用词表（约1400词）
- 扩展词汇：微信群聊场景特有词（约50词）

### 文件格式
- 编码：UTF-8
- 格式：每行一个词
- 空行自动忽略

### 修改方法

#### 1. 直接编辑文件
```bash
# 打开文件
notepad config/stopwords.txt

# 添加新词：在文件末尾添加一行
# 删除词：删除对应行
# 保存后自动生效（需重启应用）
```

#### 2. 词表分类

**已包含的停用词类型：**
- 代词：我、你、他、我们、你们、大家等
- 虚词：的、了、在、是、和等
- 副词：非常、很、比较、特别等
- 语气词：啊、哦、呢、吧、嗯等
- 时间词：今天、昨天、现在等
- 微信特有：微信、群、消息、表情、转发、@等
- 网络用语：哈哈哈、呵呵、嘿嘿等

**建议添加的词（根据实际场景）：**
- 行业专有名词
- 特定人名或品牌名
- 高频但无意义的词汇

### 技术说明

#### 加载机制
- 首次调用时加载并缓存
- 使用内存缓存提升性能
- 文件不存在时使用默认停用词

#### 过滤规则
词云分词后会过滤：
1. 长度 < 2 的词
2. 纯数字
3. 停用词表中的词
4. 群成员昵称

### 性能优化
- 停用词加载后全局缓存
- 使用set结构快速查找（O(1)复杂度）
- 推荐词表大小：300-2000词

### 故障排查

**问题1：词云仍显示某些无意义词**
- 检查停用词拼写是否正确
- 注意区分全角/半角字符
- 重启应用生效

**问题2：词云内容太少**
- 停用词过多，适当减少
- 检查聊天内容是否充足

**问题3：停用词文件加载失败**
- 检查文件路径：`project/config/stopwords.txt`
- 检查文件编码：UTF-8
- 查看控制台日志
